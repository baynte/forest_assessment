About Forest Damage Assessment

Our Mission

The Post-Typhoon Forest Damage Assessment application is designed to help environmental scientists, forestry departments, and disaster response teams quickly assess and quantify forest damage after typhoon events. By leveraging advanced image processing and machine learning techniques, we provide accurate damage assessments that can inform recovery efforts and environmental planning.

How It Works

Our system uses a comparative analysis approach:

  1. Image Collection: Users upload pre-typhoon and post-typhoon satellite or aerial images of the same forest area.
  2. Image Processing: Our algorithms process these images to identify forest areas and detect changes.
  3. Semantic Segmentation: Using deep learning techniques, we classify each pixel into categories like vegetation, water, and land.
  4. Change Detection: By comparing the pre and post images, we identify areas where forest has been damaged.
  5. Damage Quantification: The system calculates the percentage of forest area that has been damaged.
  6. Visualization: Results are presented through intuitive visualizations that highlight damaged areas.

Technology

Our application is built using:

  • Flask: A lightweight Python web framework
  • OpenCV: For image processing and computer vision tasks
  • TensorFlow: For deep learning and neural network models
  • SQLite: For database management
  • Bootstrap: For responsive and modern user interface
